Fase 02 Volgende stap

Eigen server.

Software die van jou is, verdient infrastructuur die van jou is.

Als de software van jou is maar de data in een datacenter van iemand anders draait, ben je nog steeds afhankelijk. Eigen Server betekent: een kleine, betrouwbare machine op kantoor die volledig onder eigen beheer staat. De data verlaat het pand niet, tenzij jij dat wilt.

Wat

Een server op kantoor, niet in de cloud

Een eigen server hoeft geen indrukwekkende rekenkast te zijn. Voor de meeste MKB-toepassingen is een kleine, stille computer voldoende. Vergelijkbaar met een router qua formaat, maar met de capaciteit om een compleet bedrijfssysteem te draaien.

De software die in Fase 1 gebouwd is, draait hier lokaal. Snel, betrouwbaar en volledig onder eigen beheer. Geen cloudprovider die de voorwaarden bepaalt, geen datacenter aan de andere kant van het land waar de data staat.

Eigen Grond helpt bij het inrichten van die infrastructuur als onderdeel van een project. Inclusief een back-upstrategie, zodat data nooit afhankelijk is van één apparaat of één locatie.

Hoe

Eenvoudig, robuust, betaalbaar

De hardware is bescheiden. Een kleine dedicated computer, een ononderbroken stroomvoorziening en een netwerkverbinding. Dat is het. De kosten liggen tussen de €200 en €800, eenmalig. Vergelijk dat met maandelijkse cloudkosten van €50 tot €200 die jarenlang doorlopen.

De installatie is onderdeel van het project. Bij oplevering van de software wordt de server ingericht, beveiligd en gedocumenteerd. Medewerkers leren niet alleen het systeem gebruiken, maar ook de basis van het onderhoud.

Back-ups worden automatisch aangemaakt. In de huidige situatie naar een tweede locatie via internet (versleuteld). Op termijn biedt Fase 4 een alternatief: lokale bedrijven die elkaars back-ups hosten.

Waarom

Data is macht. Locatie is controle.

Waar data staat, bepaalt wie er toegang toe heeft. Een cloudprovider kan voorwaarden wijzigen, toegang blokkeren bij betalingsgeschillen, of verplicht worden data te overhandigen aan derden. Een server op kantoor is eigendom. Fysiek, juridisch en operationeel.

De AVG (GDPR) maakt bedrijven verantwoordelijk voor waar hun data staat en wie er toegang toe heeft. Software op eigen infrastructuur maakt die verantwoordelijkheid eenvoudiger, niet complexer. Geen verwerkersovereenkomsten met drie tussenpartijen. Geen onduidelijkheid over welke data waar naartoe gaat.

En er is een praktisch voordeel: snelheid. Een systeem dat lokaal draait, reageert sneller dan een systeem dat via internet moet communiceren met een server elders. Voor dagelijks gebruik maakt dat verschil.
